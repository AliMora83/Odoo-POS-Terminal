AFRICA

// Rugged Odoo POS Terminal — Africa Edition

Three crises.
One device.
Zero downtime.

African retailers are bled by load shedding, disconnected ERPs, and predatory transaction fees. No existing solution solves all three simultaneously — until now.

01
Grid Failure
Stage 6 load shedding kills conventional POS terminals mid-transaction — no sale, no record, furious customer.
02
ERP Disconnect
Legacy hardware has zero native ERP integration. Merchants reconcile manually — wasting hours and introducing errors daily.
03
Fee Extraction
Payment processors charge 2.75% per transaction. A merchant processing $50k/month loses $1,375 — every single month.
02 / Solution

Built for Africa.
Native to Odoo.

A rugged, decoupled customer-facing terminal engineered for the African retail environment — load shedding, mobile money, and Odoo-first.

An ESP32 microcontroller drives a 2.8" TFT display and NFC reader, powered by an 18650 Li-Ion battery that survives any outage. The Antigravity FastAPI middleware connects everything directly to the merchant's Odoo ERP.

No third-party sync. No data loss. No downtime.
